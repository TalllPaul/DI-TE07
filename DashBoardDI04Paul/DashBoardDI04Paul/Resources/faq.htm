Preguntas Frecuentes

  1. ¿Como puedo salir del programa?

    2. Para cerrar el programa puedes hacer clic en 'Archivo' en el menú superior, saldrá un menú desplegable, haciendo clic en 'Salir' cerrará el programa.

  2. ¿Porque no me salen bien los datos del resumen mensual?

    3. Si no salen bien los datos del resumen mensual asegúrate que primero haces clic en el comercial y después el botón 'Resumen mensual'. Puedes encontrar más información en el documento Getting Started

  3. ¿No tengo Windows, puedo usar la aplicación?

    4. Desafortunadamente este programa ahora mismo solo está disponible para sistemas operativos Windows pero estamos trabajando para hacerlo disponible para más sistemas operativos como macOS y Linux
